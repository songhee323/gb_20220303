1. ojdbc
  1-1. jdbc (Java DataBase Connectivity)
    - 자바에서 데이터베이스에 접속/관리할 수 있게 해주는 드라이버(API).
    - 자바로 작성된 프로그램으로 DB와 관련된 작업을 처리할 수 있도록 제공되는 기능(코드)들을
	모아놓은 lib이다.

  1-2. ojdbc
    - Oracle에서 제공하는 오라클 DB 를 관리할 수 있도록 도와주는 JDBC이다.

  1-3. package
    - java.sql.Conection
	-> 특정 데이터베이스와 연결정보를 가지는 Interface이다.
	-> DriverManager 로부터 Connection 개체를 가져온다.
    - java.sql.ResultSet
	-> select 구문 실행 결과를 조회할 수 있는 방법을 정의한 Interface이다.
